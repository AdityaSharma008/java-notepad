# java-notepad

A notepad clone written in Java.

## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/AdityaSharma008/java-notepad.git
   cd java-notepad
   ```

2. **Build the project**:
   ```bash
   cd src/main/
   javac -d bin notepad/*.java
   ```

3. **Run notepad**:
   ```bash
   java -cp bin notepad.Notepad
   ```
